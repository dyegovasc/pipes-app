# Homebrew Pipes

Private Homebrew tap for Pipes apps and tools.

## Install Pipes Local

```sh
export HOMEBREW_GITHUB_API_TOKEN=your_token
brew tap dyegovasc/pipes https://github.com/dyegovasc/homebrew-pipes.git
brew install --cask pipes-local
```

`HOMEBREW_GITHUB_API_TOKEN` must have read access to the private GitHub Release asset in `dyegovasc/pipes-local`.
